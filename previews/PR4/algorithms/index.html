Lévy area simulation algorithms

Currently the following four algorithms are implemented:

  1. Fourier algorithm
  2. Milstein algorithm (Milstein, 1994)
  3. Wiktorsson algorithm (Wiktorsson, 2001)
  4. Mrongowius-Rößler algorithm (Mrongowius & Rößler, 2021)

For an in-depth discussion of the algorithms we refer to our paper Kastner & Rößler, 1994.
